Michael Harris - COP4520 Assignment 1, Problem 1 - Parallel Prime Sieve

This program was written in the latest version of Go, which can be downloaded from
	https://golang.org/dl/
	
To compile and run this program, it will first be necessary to navigate to the main.go file
location on your local machine using the terminal and cd commands. Once you have navigated to the 
correct path, you can compile and execute the source code in one step by typing the following:
	go run main.go
	
For this program, there will be no output to the screen, and output will be redirected to:
	primes.txt

The output file will be in the form of:
	<execution time> <total number of primes found> <sum of all primes found>
	<top ten maximum primes, listed in order from lowest to highest>
